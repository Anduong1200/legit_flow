  • 5
    Output Guard Streaming mode bật holdback scan và truncate an toàn nếu response có dữ liệu nhạy cảm.
  • 6
    Audit Tier 1 metadata luôn được ghi lại để đảm bảo auditability.
  • Pass: Structured PII và secrets bị block hoặc mask đúng policy và không lộ raw trên UI.
  • Pass: Streaming mode có thể truncate an toàn thay vì để response rò rỉ thông tin nhạy cảm.
  • Pass: Người xem hiểu ngay mode hiện tại, request status và request trace mà không cần đọc code trước.
  • Fail: UI quá nhiều khối ngang nhau, không có entry point rõ ràng, hoặc mô tả capability vượt quá MVP thật sự.
